### **Mux Protocol**

A fully open-source account abstraction protocol on Stellar.

**Mux** is a decentralized platform built on the Stellar blockchain that simplifies account management by abstracting smart contract interactions. With Mux, users can manage assets, interact with dApps, and automate transactions without handling multiple keys or complex workflows. Built with **Soroban smart contracts**, Mux provides a secure, transparent, and developer-friendly foundation for next-generation decentralized finance applications.
